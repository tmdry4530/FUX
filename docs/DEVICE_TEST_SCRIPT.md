# FUX 실기기 테스트 스크립트

> Android / iOS 공통, 예상 소요 시간: 15분

---

## 사전 준비

- Toss 앱 최신 버전 설치
- FUX 미니앱이 Toss 앱 내에서 접근 가능한 상태
- 네트워크 연결 상태 확인

> **주의**: `AdGate`의 `loadFullScreenAd.isSupported()`는 **Toss 앱 내에서만 `true`**를 반환합니다.
> 일반 브라우저나 WebView 디버깅 도구에서는 광고 관련 기능이 동작하지 않습니다.

## 주의 사항

1. **`isSupported()`는 Toss 앱에서만 `true`** - 일반 브라우저에서 테스트 시 광고 관련 항목은 skip
2. 출시 전 테스트 시 딥링크는 `intoss-private://` 스킴 사용
3. OG 이미지 URL은 `https://`로 시작해야 공유 카드에 표시됨
